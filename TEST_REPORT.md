## 部署边界

当前受管浏览器环境会阻止直接访问本地文件网址和自建测试域名，因此浏览器测试使用完整内嵌的单文件预览执行。真实 GitHub Pages HTTPS 环境中的 Service Worker 安装、PWA 安装提示和版本接管不能在这里直接完成。

部署包已完成资源路径、缓存清单、版本一致性和语法检查；上传后 GitHub Actions 会再次运行 `node scripts/validate-build.mjs`，通过后才自动发布。
