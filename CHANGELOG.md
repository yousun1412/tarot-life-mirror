# 更新记录

## V22.0.0

- 新增可扩展多牌组注册表与 DeckManager。
- 经典韦特78张牌面拆分为独立本地资源。
- 新增生命之镜22张大阿尔卡那上传目录与自动检测。
- 缺失大牌和全部未完成小牌自动回退经典韦特。
- 历史记录协议升级到 schemaVersion 4，保存牌组与实际回退信息。
- 牌库、牌面查看器、学习、历史、洞察和分享卡同步牌组。
- 预留卡通、写实、极简线描等未来牌组接口。
- Service Worker 对全部牌组图片按需缓存。

## V21.0.0

- 新增记录日历，可按月份浏览本日、本周、本月运势和问事抽牌。
- 新增周期回顾，为日、周、月记录补充真实发生、符合之处、修正、行动、学习和下一步。
- 新增1至5分帮助程度评分。
- 新增个人统计：常见牌、花色、正逆位、牌阵、问题领域和记录类型。
- 新增跨平台 `analytics-core`，统计逻辑可复用于手机、桌面和微信小程序。
- 阅读记录协议升级至 schemaVersion 3，兼容旧记录。
- 本地历史记录上限从100条提升到500条。
- “我的记录”和顶部记录按钮改为进入“记录与洞察”。
