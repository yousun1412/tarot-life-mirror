# 生命之镜塔罗 V13 · 完整78张 PWA

一个无需注册、不调用大模型、可安装并离线使用的本地塔罗反思应用。

## V13 核心功能

- 完整78张 Rider–Waite–Smith 塔罗牌面：22张大阿尔卡那与56张小阿尔卡那
- 正位与逆位，逆位翻牌后旋转180°
- 洗牌后牌堆始终保持叠放，不再扇形展开
- 用户从 1–78 中自行选择牌堆编号
- 三张牌模式依次选择三个不重复编号
- 本地规则牌义、三牌结构分析、牌库、历史记录和分享卡
- 首次在线打开后可离线使用

## 部署

将本目录全部文件与文件夹上传到 GitHub Pages 仓库根目录。不要只上传 ZIP，也不要遗漏 `assets/cards`、`data`、`js`、`css` 和 `icons`。

GitHub Pages 使用 `main / (root)` 发布。
