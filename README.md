# 生命之镜塔罗 V16

无需注册、可安装并离线使用的完整78张塔罗应用。

## V16重点

- 首页欢迎语精简为“欢迎来到生命之镜。‘如上，如下；如内，如外。’”
- 新增“本日运势”入口。
- 一张牌模式：今日主题。
- 三张牌模式：今日主线、今日需要留意、今日行动建议。
- 当天已有本日运势时优先回看，也可以确认后重新抽取。
- 普通抽牌继续支持自己选择1–78数字或“交给命运”随机抽取。
- 78张正逆位牌义、牌阵位置、组合分析、历史记录、分享卡和离线缓存保持可用。

## 本地预览

```bash
python -m http.server 8080
```

打开 `http://localhost:8080/`。

## GitHub Pages自动部署

仓库包含 `.github/workflows/deploy-pages.yml`。推送到 `main` 后会自动校验并发布。

首次使用时，在仓库 **Settings → Pages → Build and deployment → Source** 选择 **GitHub Actions**。
