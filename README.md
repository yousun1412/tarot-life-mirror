# 生命之镜塔罗 V17 · 本周运势

这是一个无需注册、可安装并离线使用的完整78张塔罗应用。

## V17 功能

- 普通单牌与三牌反思
- 本日运势：一张牌或三张牌
- 本周运势：三张简洁牌阵或七张完整牌阵
- 自选1–78数字，或选择“交给命运”随机抽取
- 正逆位、牌阵位置、花色、数字与组合关系解读
- 本地历史、回顾、收藏、导入导出和分享卡
- 牌图按需缓存，核心功能可离线使用
- GitHub Actions 自动校验与部署

## 本周运势牌阵

三张牌：本周整体主线—本周主要挑战—本周可用力量。

七张牌：本周整体能量—学习与工作—关系与互动—情绪与内在—现实资源—主要挑战—本周建议。

## 发布

上传整个目录到 GitHub 仓库根目录。仓库 Settings → Pages 的 Source 选择 GitHub Actions。
