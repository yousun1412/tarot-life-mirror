# 生命之镜塔罗 V12

无需注册、不调用大模型、可安装并离线使用的静态塔罗自我反思应用。

## V12 新增

- 将 V11 单文件拆分为 CSS、数据和功能脚本，便于维护
- 牌面全屏查看、缩放和正逆位切换
- 22张大阿尔卡那牌库与主题搜索
- 历史记录搜索、筛选、收藏、单条删除和后续记录
- JSON 导出与恢复
- 本地生成分享图片，默认隐藏完整问题
- 隐私说明与免责声明
- Service Worker 新版本提示和一键更新

## 发布到 GitHub Pages

将本目录中的全部文件和文件夹上传到仓库根目录。Settings → Pages → Deploy from a branch → main → /(root)。

每次升级时必须修改 `service-worker.js` 中的 `VERSION`，浏览器才会建立新缓存并显示更新提示。
