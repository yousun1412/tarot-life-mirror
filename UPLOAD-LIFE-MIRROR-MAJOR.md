# 上传生命之镜22张大阿尔卡那

## 目标目录

把22张PNG直接上传到：

```text
assets/decks/life-mirror/major/
```

不要再创建一层 `major` 文件夹，也不要上传ZIP本身。最终应类似：

```text
assets/decks/life-mirror/major/00-the-fool.png
assets/decks/life-mirror/major/01-the-magician.png
...
assets/decks/life-mirror/major/21-the-world.png
```

## 上传后

1. 提交到 GitHub `main` 分支。
2. 等待 Actions 自动检查和发布。
3. 打开网站，点击顶部菱形按钮“牌面风格”。
4. 点击生命之镜的“重新检测图片”。
5. 确认显示“检测到 22/22 张”。
6. 点击“设为默认牌组”。

未检测到的图片会自动使用经典韦特，不会导致抽牌、牌库或分享功能报错。

## 图片要求

- PNG格式
- 2:3竖版
- 建议2048×3072
- 最低建议1024×1536
- 不带透明缺口、水印或平台标识
- 文件名大小写与清单一致
